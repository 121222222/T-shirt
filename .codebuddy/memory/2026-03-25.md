## H5实时预览功能（iframe集成）
- **手机预览区改为iframe**：将原来的模拟HTML预览改为iframe直接加载 `index.html`，实现真实的H5页面预览效果
- **postMessage通信机制**：
  - `admin.js` 中新增 `npSendToIframe(action, pageIndex)` 函数，通过 postMessage 向 iframe 发送页面切换指令
  - `index.html` 中新增 `message` 事件监听器，接收 `admin-preview` 类型的消息
- **页面映射关系**：建立管理后台13个页面 ↔ H5页面的映射表 `ADMIN_PAGE_MAP`
  - 0/1/12 → `page-welcome`（欢迎页）
  - 2/3/4 → `modal-notice`（温馨提示弹窗）
  - 5 → `page-style`（选择款式）
  - 6 → `page-size`（选择尺码）
  - 7 → `page-address`（选择地址）
  - 8/9 → `page-confirm`（确认页）
  - 10/11 → `page-success`（成功页）
- **优化点击切换**：`npSelectPage()` 函数优化，只更新缩略图激活状态和右侧配置（通过 `npUpdateThumbsAndConfig()`），不重新渲染整个页面，避免 iframe 重载
- **iframe 就绪检测**：通过 `npIframeLoaded()` 回调和 `h5-preview-ready` 消息双向确认，确保 iframe 加载完成后再发送切换指令
- **演示数据填充**：当切换到尺码/确认/成功页时，自动填充演示数据以确保预览效果正常
